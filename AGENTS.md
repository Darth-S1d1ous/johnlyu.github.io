# Personal Website and Portfolio Gallery

## Introduction

Currently the site has two pages. The first page (a.k.a. page 1) is the home that demostrates my basic information, showcases results (in img and video) of my previous work, and is an entrypoint of my social media accounts. The second page (a.k.a. page 2) acts as a portfolio gallery that contains all the meaningful projects I have completed. Each entry in this gallery should have a text description of the project, and a sub-gallery that shows imgs and videos about this project.

## User Requirement Specification

- Each entry in page 2 should have transition in color when the cursor leaves, hovers on, and clicks on it.
- The sub-gallery in each entry of page 2 should allow me to scroll horizontally to view all the images and videos.
- Page 2 should allow me to add new projects, image, text descriptions with minimal effort.
- Page 1 should have an old-school style, while page 2 should be more stylish and modern.
